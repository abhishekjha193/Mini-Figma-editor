# 🎨 Mini Figma – Web Based Design Editor

Mini Figma is a lightweight web-based design editor inspired by tools like Figma and Canva.  
This project is completely built using **pure HTML, CSS, and JavaScript (Vanilla JS)** without using any external frameworks or libraries.

The application allows users to create, edit, and manage design elements such as rectangles and text on a canvas with full support for dragging, resizing, rotating, layering, and saving layouts.

---

## 📌 Project Objective

The main objective of this project is to demonstrate:
- DOM Manipulation
- Mouse & Keyboard Event Handling
- Canvas-like UI using HTML divs
- Layer management system
- Properties control panel
- LocalStorage-based save and load functionality

---

## 🚀 Features

### 🧱 Element Creation
- Add **Rectangle elements**
- Add **Text elements**

### 🖱️ Canvas Interactions
- Drag elements freely inside canvas
- Resize elements using 4 corner handles
- Rotate elements using a rotation handle
- Boundary restriction (elements stay inside canvas)

### 🧩 Layer Management
- Display all elements in Layers Panel
- Select elements directly from layers list
- Move layers **Up** and **Down**
- Automatic `z-index` management

### 🎛️ Properties Panel
- Change Width
- Change Height
- Change Background Color
- Edit Text Content (only for text elements)

### ⌨️ Keyboard Controls
- Arrow Keys → Move selected element
- Delete / Backspace → Remove selected element

### 💾 Save & Load System
- Layout is automatically saved using **LocalStorage**
- Reloading the page restores the previous design

---

## 🛠️ Tech Stack Used

- **HTML5** – Structure of the application
- **CSS3** – Styling and UI design
- **JavaScript (Vanilla)** – Complete logic and interactivity
